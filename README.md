# Web-IoT-2024
Web × IoT メイカーズチャレンジ2024年度 制作作品（仮：Nozawana）

# 環境について
Dockerを用いて環境を構築
- Ruby : 3.1
(https://hub.docker.com/_/ruby)
- MySql : 5.7
(https://hub.docker.com/_/mysql)

# 環境の実行
### スタート
```
docker-compose up -d
```
### ストップ
```
docker-compose stop
```
